# お問合せ機能 動作確認チェックリスト

## 1. 画面表示
- [ ] **お問合せフォーム画面の表示**
  - ページがエラーなく表示されること
  - デザイン・レイアウトが崩れていないこと

## 2. バリデーション（入力チェック）
- [ ] **必須項目の未入力時エラー表示**
  - 必須項目（名前、メールアドレス、お問合せ内容等）を空欄のまま送信しようとした際、エラーメッセージが表示されること
- [ ] **メールアドレス形式エラー表示**
  - メールアドレスとして不適切な形式（例: `@`がない、全角文字が含まれる等）を入力した際、エラーメッセージが表示されること

## 3. 送信・データ保存
- [ ] **送信成功時の完了画面表示**
  - 入力が正しい状態で送信ボタンを押下した後、送信完了画面（またはサンクスページ）へ遷移、もしくは完了メッセージが表示されること
- [ ] **管理者へのメール受信確認**
  - フォームから送信された内容が、管理者のメールアドレスに届いていること
  - メールの件名・本文が想定通りであること
- [ ] **MongoDBへのデータ保存確認**
  - 送信されたデータがMongoDBのコレクションに保存されていること
  - 保存されたデータの内容（名前、メールアドレス、本文、送信日時等）が入力内容と一致していること
