# 2. ユーザージャーニーマップ

## ステージ1: 認知・興味
- **行動**:
  - 英語のメールやエッセイを書いている時に、「この表現で合っているかな？」と不安になる。
  - 検索エンジンやSNSで「英語 添削 AI」「英文 チェック」などを検索する。
- **思考**:
  - 「誰かにチェックしてほしいけど、頼める人がいない」
  - 「Google翻訳だと合っているかわからない」
  - 「もっと自然な表現を知りたい」
- **感情**: 不安、焦り、向上心

## ステージ2: 利用開始（オンボーディング）
- **行動**:
  - サービスサイトにアクセスする。
  - 「AIがあなたの英文を添削・解説します」というコピーを見る。
  - アカウント登録（またはゲスト利用）をして、試しに英文を入力する。
- **思考**:
  - 「本当に使えるのかな？」
  - 「使い方は簡単かな？」
  - 「無料/安価で試せるならやってみよう」
- **感情**: 期待、少しの疑い

## ステージ3: 添削体験（コア体験）
- **行動**:
  - 自分が書いた英文を入力フォームにペースト/入力する。
  - 「添削する」ボタンを押す。
  - 数秒〜十数秒待つ。
  - AIからのフィードバック（修正案、解説、改善点）が表示される。
- **思考**:
  - 「お、速い！」
  - 「なるほど、ここはこう直せばいいのか」
  - 「解説がわかりやすい」
  - 「Claude Haiku 4.5、賢いな」
- **感情**: 納得、驚き、喜び

## ステージ4: 学習・定着
- **行動**:
  - フィードバックを読んで、自分の英文を修正する。
  - 修正した英文をメール送信や課題提出に使う。
  - 履歴機能で過去の添削内容を見返す。
- **思考**:
  - 「勉強になった」
  - 「次はもっとうまく書けそう」
  - 「また使いたい」
- **感情**: 達成感、安心感

## ステージ5: 継続利用・推奨
- **行動**:
  - 英語を書くたびにサービスを利用する（習慣化）。
  - 友人に「これ便利だよ」と勧める。
  - 有料プラン（もしあれば）を検討する。
- **思考**:
  - 「これがないと不安」
  - 「英語力が上がってきた気がする」
- **感情**: 信頼、愛着
